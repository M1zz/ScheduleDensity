개인정보 처리방침

WeekBlocks (macOS) · 최종 업데이트: 2026년 6월 8일

WeekBlocks(이하 “앱”)는 사용자의 개인정보를 소중히 여기며, 별도의 서버를 운영하지 않는 로컬 우선(local-first) 앱입니다. 본 방침은 앱이 어떤 정보를 다루고, 어디에 저장하며, 어떻게 보호하는지를 설명합니다.

핵심 요약 — 앱은 개인정보를 개발자에게 수집·전송하지 않습니다. 사용자가 작성한 데이터는 사용자 기기와 본인의 iCloud(개인 데이터베이스)에만 저장됩니다.

1.수집하는 정보

앱은 다음과 같은 개인정보를 수집하지 않습니다.

  • 이름·이메일·전화번호 등 계정/연락처 정보
  • 위치 정보
  • 광고 식별자, 사용 분석(애널리틱스), 추적 데이터
  • 연락처·사진·캘린더 등 기기의 다른 데이터에 대한 접근

앱은 별도의 회원가입이나 로그인 절차가 없습니다.

2.사용자가 작성하는 데이터

앱은 사용자가 직접 입력한 다음 데이터를 기능 제공을 위해 기기에 저장합니다.

  • 계획 블록(활동, 성공 기준, 산출물, 요일·시간대, 소요 시간)
  • 루틴(고정 시간대·주간 쿼터 설정)
  • 백로그 할 일 및 카테고리
  • 회고 기록(달성·부분 달성·건너뜀, 메모)

이 데이터는 앱의 기능을 제공하기 위한 목적으로만 사용되며, 외부로 전송되거나 제3자에게 제공되지 않습니다.

3.데이터 저장 위치

모든 데이터는 Apple의 SwiftData를 통해 사용자 기기 내부에 저장됩니다. iCloud 동기화가 켜져 있는 경우, 데이터는 사용자 본인의 iCloud 개인 데이터베이스(공유 컨테이너 iCloud.com.devkoan.ScheduleDensity)에 저장되어 같은 iCloud 계정의 기기끼리 자동으로 동기화됩니다.

iCloud를 통한 저장·전송에는 Apple의 개인정보 보호정책이 적용됩니다. 개발자는 사용자의 iCloud 데이터에 접근할 수 없습니다.

4.제3자 제공 및 외부 서비스

앱은 개발자가 운영하는 서버나 외부 분석·광고 SDK를 사용하지 않습니다. 데이터 동기화에 사용되는 Apple iCloud 외에 어떤 제3자에게도 데이터를 제공하지 않습니다.

5.데이터 보관 및 삭제

데이터는 사용자가 앱에서 직접 삭제하거나 앱을 제거할 때까지 보관됩니다.

  • 앱 내에서 개별 블록·루틴·할 일·회고를 삭제할 수 있습니다.
  • 앱을 삭제하면 해당 기기에 저장된 데이터가 제거됩니다.
  • iCloud에 동기화된 데이터는 기기의 설정 → Apple 계정 → iCloud → 저장공간 관리에서 관리·삭제할 수 있습니다.

6.아동의 개인정보

앱은 아동을 대상으로 하지 않으며, 아동으로부터 의도적으로 어떤 개인정보도 수집하지 않습니다.

7.방침 변경

본 방침은 법령이나 앱 기능의 변경에 따라 갱신될 수 있습니다. 중요한 변경이 있을 경우 본 페이지를 통해 공지하며, 상단의 “최종 업데이트” 날짜를 갱신합니다.
